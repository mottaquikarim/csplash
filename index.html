Variable Types

A variable type is a way to classify the different kinds of data we can save to a variable. There are exactly 6 types of variables:

Primitives
  • undefined
  • null
  • boolean*
  • number*
  • string*
Non Primitive
  • Object

Strings

This one is interesting, we use this to represent text. Anything between the quotations (double or single, doesn't matter as long as you are consistent) is treated as a number.

So...

var myName = 'Sam';
var gaspThisIsAString = '105';
Basic String Properties
  • Always in quotes
  • The .length property tells us how many characters in string
Useful String Methods
  • .concat
  • .slice
  • .substr
  • .substring
  • .toLowerCase
  • .toUpperCase

Nums

Numeric values that can be operated upon via the standard rules of arithmetic.

var myNumber = 1;
var pi = 3.14159; // ...approximately
  • "+" Addition
  • "-" Subtraction
  • "*" Multiplication
  • "/" Division
  • "%" Modulus (Remainder)
Example:
console.log(3)//3
console.log(3+5)//8
console.log(5*5)//25
console.log(10%2)//0
console.log(10/2)//5

Booleans

True or false. Basically.

var myBooleanValue = true; // true
var myBooleanValueThatIsFalse = false; // false
console.log( typeof myBooleanValue );
JavaScript Logical Operators
  • ===
  • !==
  • >
  • <
  • <=
  • &&
  • ||
  • !
console.log(3 === 0)//false
console.log(3 > 0)//true
console.log(3*3 === 9)//true
console.log(9%3 === 0)//true
console.log(9/3 === 0)//false

The conditional block

  • the if statement
  • the if/else statement
  • the if/else if/else statement
Example:
//This is a basic if/else statement
var score = 80; 
var grade;

if ( score < 50 ) {
    grade = "Fail";
}

else {
    grade = "Pass";
}

//this code added a third option so, we'll add an else if statement
var score = 80; 
var grade;
var incomplete = false; 

if ( score < 50 ) {
    grade = "Fail";
}
else if (incomplete === true) {
    grade = "Incomplete";
}
else {
    grade = "Pass";
}

console.log(grade);

Functions

What are functions? Functions are essentially actions. We use functions to save a set of instructions that we can tell javascript to use over and over again. Functions have names, just like variables, but they hold more than just primitive values.

//This is a function declaration.
function doSomething() { 
       alert('doing something!');
 }

Here is how we would call that function 

//declaring my function
function doSomething() { 
       alert('doing something!');
 } 
// calling it now
 doSomething();
